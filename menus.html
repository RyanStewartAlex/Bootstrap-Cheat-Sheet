Drop-down Menus

Create button with class of ".dropdown-toggle" and with attribute "data-toggle" set to "dropdown". Then make an unordered list with class of ".dropdown-menu". The list items can have the classes of ".dropdown-header", ".divider" with a role of "seperator", and ".disabled".

  • Cake
  • Games

  • Halo 3

Tabbed Menus

Click on a button and have different items show. Create an unordered list with the classes ".nav" and ".nav-tabs". Inside that unordered list create list items with links inside of them. The first must have the class ".active". and all must have the "data-toggle='tab' attribute. The links' href should be set to whatever id you want to show, i.e "#firsttab". Outside of the unordered list, create a div with the class ".tab-content". Inside of that, create divs with the ids you linked to earlier. The classes must have the classes ".tab-pane" and ".fade". The first tab to be shown must have the classes ".in" and ".active".


  • First Tab
  • Second Tab
  • Third Tab

My first tab text!

My second tab text!

My third tab text!


        <ul class="nav nav-tabs">
            <li class="active"><a data-toggle="tab" href="#firsttab">First Tab</a></li>
            <li><a data-toggle="tab" href="#secondtab">Second Tab</a></li>
            <li><a data-toggle="tab" href="#thirdtab">Third Tab</a></li>
        </ul>

        <div class="tab-content">
            <div id="firsttab" class="tab-pane fade in active"><p class="well">My first tab text!</p></div>
            <div id="secondtab" class="tab-pane fade"><p class="well">My second tab text!</p></div>
            <div id="thirdtab" class="tab-pane fade"><p class="well">My third tab text!</p></div>
        </div>

Pagination

Pagination is having and controlling page numbers on your website. Create nav with unordered list with the class ".pagination" inside of it.


  • «
  • 1
  • 2
  • 3
  • 4
  • 5
  • »

    

    <nav>
        <ul class="pagination">
            <li><a href="#" aria-label="previous"><span aria-hidden="true">&laquo;</span></a></li>
            <li><a href="#">1</a>
            <li><a href="#">2</a>
            <li><a href="#">3</a>
            <li><a href="#">4</a>
            <li><a href="#">5</a>
            <li><a href="#" aria-label="next"><span aria-hidden="true">&raquo;</span></a></li>
        </ul>
    </nav>
